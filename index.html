CSS

  • Inline CSS: applies to a specific HTML tag, using a style attribute with a CSS rulte to style a particular page element only.
  • Internal Sytle sheet: CSS rules for webpage in the head section of the HTML file
  • External CSS Style Sheet: Most common way to style webpage. Allows developers to keep all their CSS rules in a separate file, which makes design changes easier. This uses a file with a link in the head. link rel="stylesheet" href="./assets/style.css"
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element./li>
